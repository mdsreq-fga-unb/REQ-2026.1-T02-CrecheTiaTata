## Sobre o Projeto

O projeto **Creche da Tia Tata** foi desenvolvido para a disciplina de **Requisitos de Software** da **Universidade de Brasília (UnB) - Campus Gama (FCTE)**, no semestre **2026.1**.

A proposta nasce da necessidade de fortalecer a gestão interna da instituição e criar uma ponte mais eficiente entre a creche e possíveis doadores. A partir do levantamento de requisitos, análise de processos e aplicação de estratégias de engajamento, buscamos construir uma solução que apoie a sustentabilidade e amplie o impacto social da creche.

### Objetivos principais

- Organizar e centralizar informações relevantes para a gestão da instituição.
- Facilitar a conexão entre a creche e doadores em potencial.
- Oferecer visibilidade sobre demandas, recursos e necessidades.
- Apoiar a tomada de decisão com uma visão mais clara do contexto da creche.

## Escopo da Solução

### Funcionalidades previstas

- **Gestão de Requisitos**
  Documentação estruturada dos processos, necessidades e decisões do projeto.

- **Conexão com Doadores**
  Espaço voltado à aproximação entre a creche e pessoas interessadas em contribuir.

- **Dashboards de Gestão**
  Visualização de recursos, demandas e indicadores relevantes para acompanhamento institucional.

## Como Executar o Projeto Localmente

Para visualizar a documentação na sua máquina, siga os passos abaixo:

1. Clone o repositório:

```bash
git clone https://github.com/mdsreq-fga-unb/REQ-2026.1-T02-CrecheTiaTata.git
```

2. Acesse a pasta do projeto:

```bash
cd REQ-2026.1-T02-CrecheTiaTata
```

3. Instale as dependências:

```bash
npm install
```

4. Inicie o servidor local:

```bash
npm run start
```

5. Abra no navegador:

```text
http://localhost:3000
```
